# Pasta de Imagens

Coloque aqui as imagens do projeto Figma quando disponíveis.

## Imagens Necessárias

### Hero Section
- Logo SOW completo (se houver versão em imagem)
- Imagem de fundo com efeito blur (opcional)

### Mockups de Redes Sociais
- Imagens dos vídeos/mockups das redes sociais
- Logo "for mo" / Formō Paraná

### Ilustrações
- Personagem 3D com smartphone e gimbal
- Ícones customizados (se não usar emojis)

### Logos de Clientes
- Heineken
- Monster Energy
- Rock in Rio

### Logos do Grupo SD
- SD3
- Smart Data
- Adverse

## Formato Recomendado

- **Formato**: PNG com transparência ou SVG
- **Resolução**: Mínimo 2x para telas Retina
- **Otimização**: Comprimir imagens antes de usar (TinyPNG, ImageOptim)

## Uso no Código

As imagens podem ser referenciadas assim:

```html
<img src="assets/images/nome-da-imagem.png" alt="Descrição">
```

Ou via CSS:

```css
background-image: url('assets/images/nome-da-imagem.png');
```
